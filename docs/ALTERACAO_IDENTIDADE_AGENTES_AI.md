# Alteração de identidade — Agentes AI

Esta versão troca a identidade visual exibida no site de OpenClaw para Agentes AI.

## Arquivos principais alterados

- `frontend/src/app/components/Navbar.js`: logo e texto da marca na navegação.
- `frontend/src/app/components/Footer.js`: marca no rodapé.
- `frontend/src/app/layout.js`: título da aba, descrição, Open Graph e ícones.
- `frontend/src/app/icon.svg`: favicon usado pelo Next.js App Router.
- `frontend/public/favicon.svg`: favicon público.
- `frontend/public/favicon-180.svg`: ícone alternativo.
- `frontend/public/agentes-ai-logo.svg`: versão horizontal da marca.
- `frontend/public/agentes-ai-mark.svg`: símbolo compacto para navbar e favicon.

## Observação

Após publicar na Vercel, use `Ctrl + F5` ou uma aba anônima. Favicons costumam ficar presos no cache do navegador por alguns minutos.
